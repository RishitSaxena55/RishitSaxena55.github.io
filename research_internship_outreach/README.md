# Research Internship Outreach - Computer Vision

## Purpose
This folder contains personalized outreach materials for Summer 2026 research internship applications to top CV professors in the US.

## Structure
```
research_internship_outreach/
├── README.md                    # This file
├── master_email_template.md     # Base email template
├── master_resume_cv.html        # CV-focused resume
├── tracker.md                   # Outreach status tracker
│
├── MIT/
│   └── Vincent_Sitzmann/

│       ├── email.md             # Personalized email
│       └── resume.html          # Tailored resume
│
└── ... (other universities)
```

## How to Use
1. Review professor's `info.md` for their preferences
2. Customize `email.md` if needed
3. Send email with attached resume
4. Update `tracker.md` with status
